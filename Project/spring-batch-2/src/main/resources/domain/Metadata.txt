1. 스프링 배치 메타 데이터
	- 스프링 배치의 실행 및 관리를 위한 목적으로 여러 도메인들(Job, Step, JobParameters..)의 정보들을 저장, 업데이트, 조회할 수 있는 스키마 제공
	- 과거, 현재의 실행에 대한 세세한 정보, 실행에 대한 성공과 실패 여부 등을 관리함으로써 배치 운용에 있어 리스크 발생시 대처 가능
	- DB와 연동할 경우 필수적으로 메타 테이블이 생성되어야 함.

2. DB 스키마 제공
	- \org\springframework\batch\core
	- DB 유형별로 제공

3. 스키마 생성 설정
	- 수동 생성 : 쿼리 복사 후 직접 실행
	- 자동 생성 : spring.batch.jdbc.initialize-schema 설정
		* ALWAYS : 스크립트 항상 실행
		* EMBEDDED : 내장 DB일 때만 샐행되며 스키마가 자동 생성됨, 기본 값
		* NEVER : 스크립트 항상 실행 안함, 내장 DB일 경우 스크립트가 생성이 안되기 때문에 오류 발생
